01. Daily Stand-up Họp Đầu Ngày

Tình huống: Hôm qua làm xong giao diện danh sách, hôm nay làm màn hình chi tiết, nhưng bị khựng lại (block) vì backend đổi format API.

💡 Cụm từ bản xứ (Click để copy)

  • I managed to finish...
    Tôi đã xử lý/xoay sở xong... (Nghe rất chủ động)
  • I'm currently working on...
    Hiện tại tôi đang tập trung làm...
  • I've run into a minor issue with...
    Tôi vừa đụng phải một lỗi nhỏ với...
💡 Mix thì: Past Simple (Hôm qua) ➔ Present Continuous (Đang làm) ➔ Present Perfect (Vừa bị block).

✍️ Luyện tập ráp câu

Câu mẫu chuẩn:
"Yesterday, I managed to finish the list view. Today, I'm currently working on the detail view. However, I've run into a minor issue because the API response has changed."

02. Code Review Góp Ý Linh Hoạt

Tình huống: Thấy code của đồng nghiệp bị hardcode (cố định), cứng nhắc, khó mở rộng. Muốn góp ý đổi sang dùng protocol/cấu trúc linh hoạt hơn.

💡 Cụm từ bản xứ (Click để copy)

  • It's a bit too rigid / hardcoded.
    Nó hơi bị cứng nhắc / bị viết chết dữ liệu rồi.
  • What if we...?
    Sẽ ra sao nếu chúng ta...? (Đề xuất cực kỳ lịch sự)
  • Easier to maintain / extend later.
    Dễ bảo trì / mở rộng hơn về sau.
💡 Mix thì: Present Simple (Đánh giá code hiện tại) ➔ Future Simple (Sẽ dễ mở rộng hơn).

✍️ Luyện tập ráp câu

Câu mẫu chuẩn:
"I think this part is a bit too rigid because the data is hardcoded. What if we use a protocol here? It will make it more flexible and easier to extend later."
